# Präsentations-App (Streamlit)

Minimaler Einstieg für die Storytelling-Anwendung.

## Start

```bash
streamlit run presentation_app/app.py
```

## Datenquellen

- `eV-Quote_korAnalyse/datasetanalyse.csv`
- `eV-Quote_korAnalyse/counties.json`
